# 🎮 لعبة السيارات - Car Game
## نظام متكامل: HTML + CSS + JavaScript + C++

## 📋 نظرة عامة
لعبة سيارات حديثة مع واجهة ويب جميلة (HTML/CSS/JS) ونظام backend قوي بـ C++!

## 🏗️ معمارية النظام

```
┌─────────────────────────────────────┐
│     Frontend (الواجهة)              │
│  HTML + CSS + JavaScript            │
│  - تسجيل الدخول                     │
│  - اختيار السيارة                   │
│  - اللعبة (Canvas)                  │
└──────────────┬──────────────────────┘
               │
               │ حفظ في ملفات
               ▼
┌─────────────────────────────────────┐
│     Backend (المعالجة)              │
│  C++                                │
│  - قراءة البيانات                   │
│  - عرض الإحصائيات                   │
│  - إدارة المستخدمين                 │
└─────────────────────────────────────┘
```

## 🚀 كيفية اللعب

### الطريقة الموصى بها:

1. **شغل لوحة التحكم C++:**
   ```
   car_game_backend.exe
   ```

2. **اختر "1. فتح صفحة اللعبة"**
   - سيفتح المتصفح تلقائياً

3. **سجل دخول أو أنشئ حساب:**
   - إذا كان لديك حساب: سجل دخول
   - إذا لم يكن لديك: اضغط "إنشاء حساب جديد"

4. **اختر سيارتك:**
   - اختر النوع (سيدان، SUV، رياضية)
   - اختر اللون (8 ألوان)

5. **العب!**
   - استخدم الأسهم أو A/D للتحرك
   - تجنب العقبات
   - اجمع أكبر عدد من النقاط

6. **شاهد نتائجك في C++:**
   - ارجع للوحة التحكم
   - اختر "3. عرض أفضل النتائج"

## 🎯 التحكم في اللعبة

- **A** أو **←** : يسار
- **D** أو **→** : يمين
- **أزرار الشاشة** : للموبايل

## 🔄 كيف يعمل النظام؟

### 1. Frontend (JavaScript):
- يجمع بيانات المستخدم
- يحفظها في `localStorage`
- يحفظها في ملفات `.txt`

### 2. Backend (C++):
- يقرأ الملفات `.txt`
- يعرض الإحصائيات
- يدير البيانات

### 3. تدفق البيانات:
```
المستخدم → HTML Form → JavaScript → ملف TXT → C++ → عرض البيانات
```

## 🔧 لوحة التحكم C++

عند تشغيل `car_game_backend.exe`:

1. **فتح صفحة اللعبة** - يفتح المتصفح
2. **عرض المستخدمين** - قائمة جميع المستخدمين
3. **عرض أفضل النتائج** - أفضل 10 نتائج
4. **عرض إعدادات السيارة** - آخر سيارة تم اختيارها
5. **مسح جميع البيانات** - حذف كل شيء
6. **خروج**

## 🐛 حل المشاكل

**المشكلة:** الصفحة لا تفتح
**الحل:** افتح `index.html` يدوياً

**المشكلة:** البيانات لا تظهر في C++
**الحل:** تأكد من اللعب في الويب أولاً لإنشاء الملفات

**المشكلة:** الملفات لا تُحفظ
**الحل:** تأكد من السماح للمتصفح بتحميل الملفات
