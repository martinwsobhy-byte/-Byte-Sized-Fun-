# ⚙️ إعدادات المسافة بين العقبات

## 📏 كيف يعمل النظام الجديد؟

### المسافة المحكومة:
اللعبة الآن تستخدم نظام **مسافة محكومة** بين العقبات:

```
┌─────────────────────────────────────┐
│  الحد الأدنى: 180 بكسل             │
│  الحد الأقصى: 280 بكسل             │
└─────────────────────────────────────┘
```

### كيف يعمل:

1. **تتبع المسافة:**
   - اللعبة تتتبع المسافة من آخر عقبة تم إنشاؤها
   - المسافة تزيد مع كل إطار (frame)

2. **إنشاء العقبات:**
   - إذا وصلت المسافة للحد الأدنى (180): يمكن إنشاء عقبة
   - إذا وصلت للحد الأقصى (280): يتم إنشاء عقبة حتماً
   - بين الحدين: إنشاء عشوائي (30% احتمال)

3. **النتيجة:**
   - ✅ لا توجد عقبات متلاصقة
   - ✅ لا توجد مسافات طويلة جداً
   - ✅ إيقاع منتظم ومريح

## 🎮 مستويات الصعوبة

### 🟢 سهل
```javascript

MAX_OBSTACLE_DISTANCE: 350 بكسل
عدد العقبات: 1-3 في الصف
السرعة الابتدائية: 0.8x
```

**مناسب لـ:** المبتدئين، التعلم، الاسترخاء

## 🔧 كيفية تعديل الإعدادات

### الطريقة 1: من خلال الواجهة
1. افتح `difficulty_selector.html`
2. اختر مستوى الصعوبة
3. ابدأ اللعبة

### الطريقة 2: تعديل الكود مباشرة
افتح `car_game_loop.js` وعدل القيم:

```javascript
// في دالة loadDifficultySettings()
normal: {
    MIN_OBSTACLE_DISTANCE: 180,  // غير هذا الرقم
    MAX_OBSTACLE_DISTANCE: 280,  // غير هذا الرقم
    MIN_OBSTACLES_PER_ROW: 2,
    MAX_OBSTACLES_PER_ROW: 4,
    INITIAL_SPEED: 1.0
}
```

## 💡 نصائح للتعديل

### لجعل اللعبة أسهل:
- ✅ زيادة `MIN_OBSTACLE_DISTANCE` (مثلاً: 200 → 250)
- ✅ زيادة `MAX_OBSTACLE_DISTANCE` (مثلاً: 300 → 400)
- ✅ تقليل `MIN_OBSTACLES_PER_ROW` (مثلاً: 2 → 1)
- ✅ تقليل `INITIAL_SPEED` (مثلاً: 1.0 → 0.8)

### لجعل اللعبة أصعب:
- ✅ تقليل `MIN_OBSTACLE_DISTANCE` (مثلاً: 180 → 120)
- ✅ تقليل `MAX_OBSTACLE_DISTANCE` (مثلاً: 280 → 200)
- ✅ زيادة `MIN_OBSTACLES_PER_ROW` (مثلاً: 2 → 3)
- ✅ زيادة `INITIAL_SPEED` (مثلاً: 1.0 → 1.3)

## 🔍 فهم الأرقام

**البكسل إلى الوقت:**
- 100 بكسل ≈ 1 ثانية (عند السرعة 1.0x)
- 200 بكسل ≈ 2 ثانية
- 300 بكسل ≈ 3 ثواني

**السرعة:**
- 0.5x = نصف السرعة (بطيء جداً)
- 1.0x = السرعة العادية
- 1.5x = أسرع بـ 50%
- 2.0x = ضعف السرعة (سريع جداً)
