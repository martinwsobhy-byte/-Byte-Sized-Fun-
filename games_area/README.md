# 🎮 منطقة الألعاب - Games Area

## 📁 محتويات المجلد

### 🌐 ملفات الواجهة:
- `main_menu.html` - الصفحة الرئيسية لمنطقة الألعاب
- `difficulty_selector.html` - صفحة اختيار مستوى الصعوبة
- `car_game_loop.html` - صفحة اللعبة الرئيسية
- `car_game_loop.css` - تنسيق اللعبة
- `car_game_loop.js` - منطق اللعبة

### ⚙️ ملفات الإعدادات:
- `game_settings.js` - إعدادات اللعبة القابلة للتعديل

### 📚 ملفات التوثيق:
- `GAME_INSTRUCTIONS.md` - تعليمات اللعبة
- `DISTANCE_SETTINGS.md` - شرح إعدادات المسافة
- `README.md` - هذا الملف

## 🔄 تدفق اللعبة:

```
تسجيل الدخول
       ↓
منطقة الألعاب (main_menu.html)
       ↓
اختيار الصعوبة (difficulty_selector.html)
       ↓
اختيار السيارة (car_game_loop.html)
       ↓
اللعب (Canvas Game)
       ↓
العودة لمنطقة الألعاب
```

## 🎯 المسارات:

### من خارج المجلد:
- `../games_area/main_menu.html` - للدخول لمنطقة الألعاب

### داخل المجلد:
- `difficulty_selector.html` - اختيار الصعوبة
- `car_game_loop.html` - اللعبة
- `main_menu.html` - العودة للقائمة

## 🎮 الألعاب المتاحة:

### 🏎️ لعبة السيارات:
- **5 حارات** للعب
- **4 مستويات صعوبة** (سهل، متوسط، صعب، خبير)
- **مسافات محكومة** بين العقبات
- **نظام نقاط** متقدم

### 🚀 ألعاب قادمة:
- لعبة الفضاء
- لعبة الألغاز
- المزيد قريباً...

## 🔧 للمطورين:

### إضافة لعبة جديدة:
1. أنشئ ملفات HTML/CSS/JS للعبة
2. ضعها في مجلد `games_area`
3. أضف بطاقة اللعبة في `main_menu.html`
4. اربط المسارات

### تعديل الإعدادات:
- عدل `game_settings.js` لتغيير إعدادات اللعبة
- عدل `car_game_loop.js` لتغيير منطق اللعبة
