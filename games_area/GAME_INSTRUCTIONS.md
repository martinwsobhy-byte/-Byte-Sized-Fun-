# 🎮 تعليمات اللعبة المحدثة

## ✨ التحديثات الجديدة

### 🛣️ نظام الحارات الجديد:
- **5 حارات** بدلاً من 3
- **ضمان وجود حارة فارغة دائماً** - لن تجد نفسك محاصراً!
- **توزيع أفضل للعقبات** - من 2 إلى 4 عقبات في كل صف

### 🎯 كيف يعمل النظام:

```
┌─────┬─────┬─────┬─────┬─────┐
│  1  │  2  │  3  │  4  │  5  │  ← 5 حارات
├─────┼─────┼─────┼─────┼─────┤
│  ❌  │  ❌  │  ✅  │  ❌  │  ✅  │  ← دائماً حارة واحدة فارغة على الأقل
├─────┼─────┼─────┼─────┼─────┤
│  ✅  │  ❌  │  ❌  │  ✅  │  ❌  │
├─────┼─────┼─────┼─────┼─────┤
│  ❌  │  ✅  │  ❌  │  ❌  │  ✅  │
└─────┴─────┴─────┴─────┴─────┘

❌ = عقبة
✅ = حارة فارغة (يمكنك المرور)
```

## 🎮 التحكم:

### لوحة المفاتيح:
- **A** أو **←** : الانتقال لليسار (حارة واحدة)
- **D** أو **→** : الانتقال لليمين (حارة واحدة)

### الشاشة:
- **⬅️** : زر اليسار
- **➡️** : زر اليمين

## 📊 نظام اللعب:

### الحارات:
```
الحارة 1 (أقصى اليسار)
الحارة 2
الحارة 3 (الوسط) ← تبدأ هنا
الحارة 4
الحارة 5 (أقصى اليمين)
```

### العقبات:
- **عدد العقبات في كل صف:** 2-4 عقبات
- **الحارات الفارغة:** 1-3 حارات
- **ضمان:** دائماً يوجد طريق للمرور!

## 🏆 استراتيجية الفوز:

1. **راقب الطريق أمامك** - انظر للعقبات القادمة
2. **خطط مسارك** - اختر الحارة الفارغة مبكراً
3. **لا تتسرع** - الحركة السلسة أفضل من العشوائية
4. **استخدم كل الحارات** - لا تلتزم بحارة واحدة

## 🎯 نصائح:

✅ **افعل:**
- تحرك بين الحارات بسلاسة
- راقب السرعة - تزيد كل 500 نقطة
- استخدم الحارات الجانبية عند الحاجة

❌ **لا تفعل:**
- لا تبقى في نفس الحارة طوال الوقت
- لا تتحرك في اللحظة الأخيرة
- لا تنسى أن السرعة تزيد!
